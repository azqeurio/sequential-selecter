# 시퀀셜 셀렉터 (Sequential Selector)

시퀀셜 셀렉터는 사진을 빠르게 비교하고 선별하기 위한 데스크톱 애플리케이션입니다. 폴더를 선택하면 폴더 안의 모든 지원되는 이미지가 썸네일로 나타나며, 클릭이나 단축키를 통해 두 개의 프리뷰 슬롯에서 이미지를 비교할 수 있습니다. 이미지 파일은 드래그 & 드롭 또는 단축키로 타겟 폴더로 이동할 수 있어 비슷한 사진을 효율적으로 걸러낼 수 있습니다.

## 주요 기능

* **그리드 썸네일 뷰**  – 선택된 폴더의 이미지가 정사각형 썸네일로 표시되며 Ctrl+마우스 휠을 사용해 썸네일 크기를 확대/축소할 수 있습니다. 확대하면 자동으로 더 큰 해상도의 썸네일을 불러와 품질을 유지합니다.

* **프리뷰 슬롯**  – 두 개의 독립적인 프리뷰 영역에서 사진을 확대/축소하며 비교할 수 있습니다. 슬라이더로 확대 비율을 조절하거나 마우스 휠로 직접 조절할 수 있으며, 확대/스크롤 상태는 다른 사진을 열어도 유지됩니다.

* **드래그 & 드롭** – 선택된 이미지를 하단의 Target 버튼으로 끌어다 놓아 다른 폴더로 이동할 수 있습니다.

* **키보드 단축키** – 방향키를 이용해 썸네일 간 이동하고 Enter 키로 현재 선택된 사진을 프리뷰에 표시할 수 있습니다. Ctrl+Z로 마지막 이동을 되돌릴 수 있습니다. 숫자 키 1/2를 사용해 선택한 이미지를 각각 Target1/Target2 폴더로 신속하게 이동할 수 있으며, 키를 누른 상태에서 썸네일을 클릭하면 단일 파일도 바로 이동합니다.

* **듀얼 모드** – Ctrl+D 키나 “듀얼 모드” 버튼을 사용해 썸네일 리스트와 프리뷰 창을 독립된 두 창으로 분리하거나 다시 합칠 수 있습니다. 듀얼 모니터 환경에서 특히 유용합니다.

* **Undo 기능** – 실수로 파일을 잘못 이동했을 때 Ctrl+Z로 직전 이동을 되돌릴 수 있습니다.

* **드래그 선택 및 미리보기 개선** – 러버 밴드로 여러 썸네일을 한 번에 선택할 수 있고, 드래그 프리뷰가 향상되어 선택 개수가 표시됩니다.




### 1. 사전 요구 사항

windows 에서는 release에서 exe 받아서 쓰면됩니다.

macOS에서 빌드를 진행하기 위해 다음 항목이 필요합니다.

- macOS 12 이상 (Apple Silicon 권장)
- Python 3.10 이상  
  (Homebrew 또는 python.org의 **universal2** 버전 권장)


### 2. 개발 환경 준비

아래 명령어를 순서대로 실행하여 필수 패키지를 설치합니다
```
python3 -m pip install --upgrade pip
python3 -m pip install PySide6 rawpy pillow pillow-heif
python3 -m pip install pyinstaller
```
### 3. .app 파일 생성

asp.py가 있는 프로젝트 디렉터리로 이동한 후 다음 명령을 실행합니다:
```
pyinstaller \
  --name "Sequential Selector" \
  --windowed \
  sqs.py
```
